# Bellabeat Case Study 📱

A data analytics capstone project analyzing smart device usage data to identify trends and provide marketing recommendations for Bellabeat, a health-focused technology company.

## Business Task
Identify trends in how consumers use non-Bellabeat smart devices to apply insights into Bellabeat's marketing strategy.

## Tools Used
- **Excel** — data cleaning and organization
- **SQL (BigQuery)** — data processing and analysis
- **Tableau** — data visualization

## Key Findings
- Active hours contribute to burning calories but are not the only factor
- No direct relationship found between sleep hours and calories burned
- Steps per day is a significant factor in calorie burn
- 21 out of 33 users tracked activity for the full 31 days

## Recommendations
- Build an app to track client activity and nutrition
- Add reminders for hydration and sleep schedules
- Offer a 2-month free trial to build user trust and habit
